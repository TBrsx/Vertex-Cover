DESCRIPTION : Dans le cadre de l'unité d'enseignement COMPLEX du M1 d'informatique de Sorbonne Université, j'ai proposé des méthodes de résolution du problème VERTEX COVER, comprenant des méthodes de branch&bound.

USAGE: 
Lancer " main.py "fichier.txt" " dans un terminal, avec fichier.txt un fichier contenant un graphe tel que exempleinstance.txt
Si aucun argument n'est donné, le programme s'exécute sur les 4 exempleinstance.txt contenu dans le dossier

CONTENU :
- graphes.py : fonction de bases
- approchees.py : méthodes approchées
- BB.py : différents algorithmes Séparation - Évaluation
- main.py : Fonction à apeller pour faire tourner les algos sur un exemple
- tests.py : Très moche, sert à générer les diverses courbes utilisées (ou non) dans le rapport rendu pour évaluation
- 4 exemples d'instances sous format txt
